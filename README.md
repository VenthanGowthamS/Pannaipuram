# பன்னைபுரம் App
### உங்கள் ஊரின் தகவல் மையம் — Your Village's Information Centre

A hyperlocal Android app built for the people of Pannaipuram, Tamil Nadu.

## The 5 Modules

| Module | Tamil | Purpose |
|---|---|---|
| ⚡ Electricity | மின்சாரம் | TNEB power cut alerts, fault reporting |
| 💧 Water | தண்ணீர் | Community water arrival alerts |
| 🚌 Bus | பேருந்து | Offline bus timetable — Thevaram, Uthamapalayam |
| 🏥 Hospital | மருத்துவமனை | PTV Padmavathy Hospital doctor timings |
| 📞 Emergency | அவசர தொலைபேசி | One-tap emergency contacts |
