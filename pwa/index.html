என்ன இது?

பண்ணைப்புரம் — தேனி மாவட்டத்தில் உள்ள நம்ம ஊர். இந்த app-ல பேருந்து நேரம், ஆட்டோ தொடர்பு, மற்றும் ஊர் சார்ந்த தகவல்கள் எல்லாம் ஒரே இடத்தில் கிடைக்கும்.

What is this?

A Tamil-first village information app for Pannaipuram, Theni District. Bus timings, auto contacts, and essential local info — all in one place, works offline.

யார் செய்தது?

வெந்தன் (Venthan) — நம்ம ஊர் பையன். ஊர்ப் பிரச்சனை தீர்க்கணும்னு உருவாக்கப்பட்டது.

முக்கிய features

  • 🚌 எல்லா பேருந்து நேரமும் — next bus countdown
  • 🛺 ஊர் ஆட்டோ டிரைவர் தொடர்பு
  • 📴 Offline-ல வேலை செய்யும்
  • 🔄 Auto-refresh (every 10 min)
  • 💬 Feedback admin-க்கு நேரடியா
📖 எப்படி பயன்படுத்துவது
  1. 1

    🚌 பேருந்து நேரம் பார்க்க

    கீழ இருக்கும் பேருந்து tab-ஐ அழுத்துங்க. எல்லா routes-ம் வரும் — அடுத்த பஸ் எத்தன நிமிடத்தில என்று clearly தெரியும்.

  2. 2

    🔍 குறிப்பிட்ட இடம் தேட

    மேல உள்ள search box-ல "தேனி", "போடி", "கம்பம்" என்று type பண்ணுங்க. உடனே filter ஆகும்.

  3. 3

    📋 முழு timetable பார்க்க

    எந்த route card-ஐயும் அழுத்துங்க — அன்றைய எல்லா பேருந்து நேரமும் expand ஆகும். நீண்ட gap இருந்தா warning-ம் வரும்.

  4. 4

    🛺 ஆட்டோ/வேன் அழைக்க

    கீழ ஆட்டோ tab-ஐ அழுத்துங்க. ஒவ்வொரு driver-க்கும் பக்கத்தில 📞 அழைக்க button — ஒரே tap-ல call போகும்.

  5. 5

    🔄 புதுப்பிக்க

    ஆட்டோ tab-ல மேல-வலது ↻ icon அழுத்தினா புதுசா update ஆகும். Bus timings-ம் ஒவ்வொரு 10 நிமிடத்துக்கும் auto-refresh ஆகும்.

  6. 6

    📱 Home screen-ல install பண்ண

    Browser menu-ல "Add to Home Screen" அல்லது "Install app" select பண்ணுங்க. அடுத்தமுறை icon tap பண்ணினா app-ஆ திறக்கும் — வேகமா, offline-ம்.

  7. 7

    💬 பிரச்சனை வந்தா

    மேல-இடது ☰ menu → கருத்து சொல்லுங்க → உங்க பிரச்சனை type பண்ணுங்க. நேரடியா admin-க்கு வரும்.
